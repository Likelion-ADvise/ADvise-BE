# 멋쟁이 사자처럼 해커톤
멋쟁이 사자처럼 해커톤에서 사용한 서버코드입니다. Django를 사용하여 구현하였습니다.

## 👨‍🏫 프로젝트 소개
**ADvise**는 광고 역경매 서비스로, 업체들이 광고를 제시하면 인플루언서 / 광고사들이 역경매로 가격을 제시하여 합리적인 가격으로 광고를 할 수 있게 도와주는 서비스입니다.

## ⚙️ 기술 스택

* WS/WAS: Nginx
* 아이디어 회의: Notion

## 📌 주요 기능
* 광고 작성
* 광고 검색
* 댓글 작성
* 댓글 조회
* 댓글 삭제
